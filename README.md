# MBTI 性格测试

基于 [Open Extended Jungian Type Scale (OEJTS)](https://openpsychometrics.org/tests/OEJTS/) 的纯前端 MBTI 性格测试。

## 特性

- **OEJTS 32 题版**：基于科学开源量表
- **约 5 分钟**：单题模式，轻松作答
- **隐私安全**：数据仅存本地，零追踪
- **开源无广告**：纯前端实现

## 在线体验

[https://mbti.daum.pw](https://mbti.daum.pw)

## 技术栈

- HTML + CSS + JavaScript（纯静态）
- 部署于 Cloudflare Pages

## 题目来源

Open Extended Jungian Type Scale (OEJTS) 1.2
